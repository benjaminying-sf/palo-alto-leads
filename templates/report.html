📋 5-Minute Manual Source Checklist

The automated scrapers run every Sunday but the county websites block robot access. These 3 links take under 5 minutes total each week — then use python main.py --import-csv FILE --type TYPE to add any leads you find.

1. PROBATE — Santa Clara Superior Court

Search for new probate cases filed in the last 7 days.

Open Court Portal →

Steps: Click "Case Type Search" → Case Type: Probate (PR) → Date Range: last 7 days → Search. Export or copy the list. Download as CSV if available, then: python main.py --import-csv FILE --type probate

2. PRE-FORECLOSURE (NOD) — Santa Clara Recorder

Search for Notices of Default recorded in the last 7 days.

Open Recorder Search →

If the website still doesn't work online, visit the office in person: 70 W. Hedding St., San Jose, CA 95110 (Mon–Fri 8am–4pm). Ask for "NOD/NTS documents recorded in Palo Alto this week." Or: search PropStream free trial for the week.

3. TAX DELINQUENT — DTAC Auction List

Check if any Palo Alto properties are on the current tax-default auction list.

Open DTAC Auction Page →

The county posts a new list before each annual auction (usually Feb–May). If there's a downloadable CSV/Excel, download it and run: python main.py --import-csv FILE --type tax_default

BONUS: BANKRUPTCY FILINGS (automated if CourtListener token is set)

Chapter 13 bankruptcy = homeowner trying to stop foreclosure = highly motivated seller. This source runs automatically if you've added a free CourtListener token to .env. Setup (2 min): Go to courtlistener.com/register → free signup → Profile → API Keys → Add Token → paste in .env as COURTLISTENER_TOKEN=your_token

I work with homeowners who'd rather sell privately and walk away with cash in hand rather than go through foreclosure. If you're open to it, I can offer a quick, no-obligation cash offer. Would you have 10 minutes to talk?"
